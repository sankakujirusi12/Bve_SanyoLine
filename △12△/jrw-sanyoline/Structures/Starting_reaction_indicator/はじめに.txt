『使用について』
このレピーターは本体であるStarting reaction indicator.xとSignalのx,y,z座標を同じにしていただくと、見た目よく使用できます。
また使用する際はsectionの値を必ず1にしてください。
使用例）
Structure['Starting'].Put('', -1.5, 3, 0, 0, 0, 0, 0, 0);
Signal['type'].Put(1, '', -1.5, 3, 0, 0, 0, 0, 0, 0);


このレピーターは位置調整を全くしていません。(使用者がどんなホームか分からないため)

そして反対側から見ると光ってません。（手間なので）あくまで進行方向だけ光ります。

色や形がおかしい場合、個人で改造しても結構ですが、教えてくださると幸いです。連絡先はこの.zipファイルをダウンロードしたお問い合わせです。（色は絶対に違います…）

あくまでこれは次の信号を表しているので、必ず後に信号機を置いてください。

このレピーターは画像や動画の投稿、または改造可といたしますが、単体での公開は禁止いたします。ルールを守ってご使用ください。（連絡先：お問い合わせ）
