[仕様＆実車との相違]
先頭がT車 = Ver5.5対策です。
ブレーキ = 本体の自動ブレーキが機関車を想定したものでは無いので。
HB動作時のブレーキ = そこまでやる必要は無いかと。

[特徴]
実は高速走行が大の苦手で、ブルートレインや高速貨物には全く向いていない。
低速での牽引力を生かして最高速65～75kmの専貨でのんびり走るのに最適。

[運転法]
ノッチは捨て4段・S・SP・P・WF1～4の全11段で、S・SP・Pは自動進段。
ノッチアップは必ず進段ランプが消えてから行ってください。弱め界磁の限流値は500Aで。
捨てノッチ以外は保ちが可能。ただしS・SP・Pでは電流計が、設定した限流値を指している時は
抵抗が抜けていないので、その状態をあまり長く保たないようにして下さい。
捨てノッチで、ある程度(1～3メートル)進めたら後は速やかに上の段に投入して下さい。
なお自動進段中(ノッチ進め灯点灯時)のノッチアップはHB動作につながるので注意。

[プラグイン]
PageUp(デフォルト) = 限流値調整(上ゲ)…ノッチオフ時に使用可能。
PageDown(デフォルト) = 限流値調整(下ゲ)…ノッチオフ時に使用可能。
D2(デフォルト) = 前照灯(のようなもの)…実際に明るくなる訳ではありません。
D3(デフォルト) = ハイビーム(前照灯が点いている時のみ動作)。
D5(デフォルト) = 弱め界磁レバー(アップ)…SP最終段で動作します。
D6(デフォルト) = 弱め界磁レバー(ダウン)…SP最終段で動作します。
D7(デフォルト) = マスコンの握り…押している間は自動進段を停止します。
D8(デフォルト) = HBリセット…HBは自動進段中のノッチアップや各段の800Aを超えたあたりで動作します。
D0(デフォルト) = ブロワー入切…ランプと音だけです。

HBの復帰はノッチを戻して、リセットすれば復帰します。
弱め界磁レバーは、進段知らせ灯が消灯してから扱ってください。
限流値調整は450,500,550,600のみ搭載(600以上はあまり意味が無いので)。
デフォルトは500にセットされています。
機構的にはノッチオフ時ならいつでも使用可能ですが、
規則面で走行中の取り扱いが許可されていたかどうかは不明。
